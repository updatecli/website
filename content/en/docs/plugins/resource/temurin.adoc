== Description

**source**

The Temurin (Adoptium) "source" retrieves a release name, Installer URL, Checskum URL or signature URL from Adoptium (link:https://adoptium.net/[]) using their API (link:https://api.adoptium.net/[]).

**condition**

Unsupported

**target**

Unsupported
